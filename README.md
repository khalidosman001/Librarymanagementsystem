 Online Library Management System

Description
This project implements an Online Library Management System using Object-Oriented Programming in C++.

 Features
- Add and remove books
- Search books by title
- Borrow and return books
- User and Book encapsulation

 Tools Used
- Code::Blocks IDE
-  GCC Compiler

How to Run
1. Open the project in Code::Blocks
2. Build and Run the project (F9)
